# EGS Coursework – Case A: Smart Home Energy Management

## Overview
This repository contains the Python implementation of the energy system model developed for the EGS coursework.

The model simulates energy flows between PV generation, battery storage, household load, and the grid under time-varying tariffs.

## Contents
- caseA_model.py: main simulation model

## Method
The model includes:
- Energy balance equations
- Battery SOC tracking
- Grid import/export logic
- Cost calculation

## How to Run
Run the Python script:

```bash
python caseA_model.py
